## 設計哲學
- 任務 UI **填進 v1.1-0 既有槽位**（TaskSlot=任務卡、CenterSlot=箭頭/光圈/慶祝），不另起 HUD。
- 慶祝沿用「一次性 gate + 收集簿可重看」哲學（小司機 J）。

## 硬性約束
- 任務卡→`TaskSlot`、目標箭頭/光圈→`CenterSlot`、達成揭曉/慶祝→`CenterSlot` overlay；**不新增槽位**。
- 「台北飛透透」大慶祝＝**一次性 gate**（`tp_celebrated`），之後從收集簿重看。
- 地標點亮**全家共享**（雙人誰點亮都算）。

## 既有資產照抄
- 槽位：v1.1-0 `TaskSlot/CenterSlot/StatusSlot`。
- 浮名牌：`src/display/render/labels.js`。
- 慶祝動效/煙火：玩具櫃 milestone 慶祝精神（web 重做）+ v1.1-0 Tactile/success 粒子。
- 達成事件：v1.1-3 任務判定、v1.1-2 迫降成功。

## 分 Phase 實作
**P1 — 任務卡 + 目標導引**：TaskSlot 顯示當前任務 prompt（DRAFT）；CenterSlot 目標方向箭頭 + 距離 + 進場光圈（地標尋寶/穿圈）。

**P2 — 達成揭曉 + fact 浮名牌**：達成 → CenterSlot success（粒子/Settle）+ 地標旁浮 fact 名牌（DRAFT）。

**P3 — 收集簿 + 點亮 + localStorage**：點亮過的地標/完成任務 → 收集簿頁；`tp_landmarks_lit` / `tp_missions_done`（沿用 spec §7）；雙人共享點亮。

**P4 — 台北飛透透大慶祝**：全地標點亮 → 全屏慶祝（夜景+煙火，CenterSlot overlay）；`tp_celebrated` 一次性；收集簿可重看。

**P5 — 測試 + HITL**：
- vitest：點亮/收集狀態、一次性 gate 邏輯、收集簿讀取、雙人共享點亮。
- e2e：達成任務→揭曉→點亮→（湊齊）大慶祝→重看。
