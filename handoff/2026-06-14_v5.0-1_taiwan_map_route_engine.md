- A→B 跨海不渲染 300km → 全圖抽象（2D 選線 + 雲上 3D + 時間壓縮）。全圖**順便＝台灣地理學習地圖**（9 機場標位置，本島/離島/東部一目了然）。

## 硬性約束（拍定）
- **全圖巡航 hybrid**：2D 選線地圖 + 雲上低細節 3D 巡航 + 時間壓縮；**巡航＝半自動**（玩家可微調航向、近目的地恢復全控）。
- **airport-template**：每機場一個 airspace，參數化（runway / taxiway(V4) / weatherProfile(V3) / landmarks / gates）；換空域＝換資料、引擎零改動（B1）。
- 機場切換：load 到達 / unload 出發 airspace。
- **3 demo 機場**：松山(songshan) + 高雄(khh) + 金門(knh)——涵蓋 本島↔離島 + 霧招牌。其餘 6 機場走內容滴（v5.1-2/ongoing）。
- 美術 **§11**：機場/設備 props＝CC0 low-poly（spike-glb recipe）；台灣地標＝手刻 voxel。
