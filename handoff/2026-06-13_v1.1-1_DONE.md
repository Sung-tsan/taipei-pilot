## 設計決定（附「為何」）
- **忘放輪（justNoGear）所有模式只提醒彈開、不扣血/不受損**：那是操作提醒不是撞擊，扣血太苛。撞擊後果只認「建築撞擊 / 非起落架的地形墜撞」。
- **smoking 僅視覺＋警告、不降性能**（handoff 拍定；降性能留 backlog）。冒煙視覺＝機尾深煙脈動 + StatusSlot ⚠️ + CenterSlot toast。
- **齒輪入口 z-index 提到 11（高於 joinPanel 10）**：家長要能在「等小孩連線」的 join 畫面就先設好後果模式。
- **damage state machine API 乾淨**（`registerMishap` 回 outcome+reset）：v1.1-2 劣質迫降→受損、V5 油盡→迫降可直接複用 `smoking/destroyed`（接口已備好）。
