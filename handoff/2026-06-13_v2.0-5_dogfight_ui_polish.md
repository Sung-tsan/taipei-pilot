## 設計哲學
- 武器 UI **填 v1.1-0 既有槽位**（鎖定框→CenterSlot、彈藥/冷卻→StatusSlot、擊落數/命中率→計分槽），不另起 HUD。
- remote 武器鍵填 `context-keys.js` 的 2 顆替換鍵（DOGFIGHT_KEYS）。
- 音效照 tone ladder：卡通飛彈可愛音、空對空/地擬真爆炸（hybrid：擬真走取樣 clip）。

## 硬性約束
- 不新增 HUD 槽位/不改協定；武器鍵走 context-keys config。
- 擬真爆炸用 one-shot 取樣 clip（合成做不出質感）；卡通飛彈合成。

## 既有資產照抄
- HUD 槽位：`src/display/ui/{hud,hud-slots}.js`。
- remote 鍵：`src/remote/context-keys.js`（DOGFIGHT_KEYS：發射 momentary + 武器切換 action）。
- 音效：`src/display/audio.js`（擴充 + 引入 clip 載入）；fx 通道 `src/display/net/client.js`。
- 鎖定/計分 state：v2.0-2/-3（本階段做呈現）。

## 分 Phase 實作
**P1 — 空戰 HUD**：鎖定框（CenterSlot，鎖定中/已鎖視覺）、彈藥+冷卻（StatusSlot）、擊落數+命中率計分（計分槽）、子模式標示（ModeSlot）。分屏每半屏一套。
**P2 — remote 武器鍵**：DOGFIGHT_KEYS＝發射（momentary，按住/點）+ 武器切換（卡通/空對空/空對地循環）；簡單版 2 大鍵、複雜版可多一排（沿用 complex-controls）。
**P3 — 音效**：發射（卡通「咻噗」/擬真）、命中（叮/啵）、擊落爆炸（擬真取樣）、鎖定提示音。hybrid 落地。
**P4 — 整合 + 教學**：空戰各子模式接模式選單；首次教學瞬間（純視覺：鎖定→發射）；回歸測試。
**P5 — 雙真機 HITL 全項**（Sung 親驗、不可 proxy）。
