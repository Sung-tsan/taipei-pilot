## 設計哲學
- 這是 **UI 層的 B1**：設計一次 seam，V1–V5 後面填槽位不重畫。
- 維持玩具櫃/voxel-toybox 識別（Tactile web 版），把 UI 系統化——**不換風格**。
- display＝固定槽位 contextual HUD（每模式只亮要的）；remote＝底層控制 + 每模式可換 context 動作鍵；簡單/複雜兩前端共用同一協定。

## 硬性約束
- **不碰飛行模型／場景／連線核心**（`flight-model.js` / `scene/` / `server/` / `shared/` 連線邏輯）。本輪只動「呈現層 + 控制輸入層 + 協定欄位擴充」。
- 協定擴充必須**向後相容**：簡單版送既有欄位、複雜版多送 `rudder/trim/flaps`；display 對缺欄位以中立值處理（舊 remote 仍可連）。
- 不引新前端框架/依賴（vanilla + Three.js + 現有結構）。
- 色票/圓角/Tactile press/haptic 政策集中在新設計語言檔，不散落。

## 既有資產照抄（reuse，勿重寫）
- display HUD：`src/display/main.js`、`src/display/render/{viewports,labels,chase-cam}.js`、`src/display/ui/qr-panel.js`（回家箭頭/高度帶/QR）。把現有元素**搬進槽位**，不重畫。
- remote：`src/remote/{main,tilt,throttle,feedback}.js`（校準/油門/傾斜重力投影/震動閃屏）。簡單版＝現況歸位。
- 協定：`shared/protocol.js`（`{slot,roll,pitch,throttle,btn:{land,horn}}` + 既有 `pstate` 機況回傳）。

## 分 Phase 實作
**P1 — 玩具櫃 web 設計語言檔 `taipei_pilot/DESIGN_WEB_UI.md`**
- 色票（草地 `#9DBF7B` 系/河流粉藍/糖果色/101 青綠/暖米背景）、圓角、Tactile press（CSS transform 形變＋微暗＋回彈，重做 DESIGN_CRAFT_GUIDE 精神）、haptic 政策（Android vibrate／iOS 閃紅框＋音效）。
- **HUD 槽位定義**：`TaskSlot`(頂中)、`StatusSlot`(左上)、`ModeSlot`(右上)、`CenterSlot`(中央)、`AltBand`(右緣)、`HomeSlot`(左下)；每槽 contextual 顯示契約（哪模式亮什麼）。

**P2 — display HUD 框架**
- 抽出 6 槽位元件 + contextual 顯示控制（`mode → 槽位內容` map）。
- 把 v1 既有元素搬進槽位（回家箭頭→HomeSlot、高度帶→AltBand、機種→ModeSlot）；TaskSlot/StatusSlot/CenterSlot 先留空契約（v1.1-1/-4 填）。
- 分屏每半屏一套槽位（`viewports.js` 分屏不動，HUD 改 per-viewport 掛載）。

**P3 — remote 簡單版框架**
- 底層：油門豎滑桿＋傾斜＋姿態泡泡＋校正＋起落架；**context 動作鍵區（2 顆，預設 降落/喇叭）抽成可替換 slot**。
- 橫拿/校準/軸向沿用現況（**不動 `tilt.js` 重力投影**）。

**P4 — remote 複雜版 + 協定擴充**
- 加：迷你儀表（空速/高度/航向/姿態，走既有 `pstate` 回傳通道擴充）、方向舵（`rudder`→yaw）、襟翼（`flaps` 分段）、配平（`trim`）。
- 開始操控前 scheme 選擇（簡單/複雜），存 `tp_remote_scheme`（per device）。
- `shared/protocol.js` 加 `rudder(-1..1)`、`flaps(0..n)`、`trim`，**向後相容**（缺值＝中立）。display 接 `rudder→yaw`、`flaps→升力/阻力`、`trim→配平偏置`（接 flight-model 既有輸入，最小改動）。

**P5 — 測試**
- vitest：協定向後相容序列化、槽位 contextual 顯示邏輯、scheme 切換、rudder/flaps/trim 值域。
- Playwright e2e：①**舊式 remote（無新欄位）仍可連飛**（向後相容回歸）②複雜版 scheme 送 rudder ③分屏每半屏槽位渲染。
